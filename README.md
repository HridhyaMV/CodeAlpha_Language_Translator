# Universal Language Translator

## Introduction

Universal Language Translator is a web-based application developed using Python and Flask. It allows users to translate text between multiple languages through a simple and user-friendly interface.

## Features

* Translate text between multiple languages
* Text-to-Speech functionality
* Copy translated text
* Language swapping feature
* Simple and responsive interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Google Translator (deep-translator)
* gTTS

## How It Works

1. Enter text in the input box.
2. Select the source language.
3. Select the target language.
4. Click the Translate button.
5. View the translated text.
6. Use the copy or speak option if needed.

## Project Structure

* app.py
* templates/index.html
* static/style.css
* requirements.txt

## Conclusion

This project demonstrates the use of language translation and text-to-speech technologies in a Flask-based web application.
